# 🐱 哆啦AI梦日报归档系统

基于GitHub Pages的日报归档网站，自动部署，永久保存。

## 🌐 网站功能

### 核心功能
- **AI行业日报** - 每日10:00推送，宏观AI技术前沿
- **游戏OC日报** - 每日10:30推送，游戏公司OC专属资讯
- **完整归档** - 所有日报永久保存，随时回看
- **响应式设计** - 手机、平板、电脑全适配

### 技术特点
- **静态网站** - 纯HTML/CSS/JavaScript，无需后端
- **自动部署** - GitHub Actions自动构建和部署
- **全球CDN** - GitHub Pages全球加速
- **完全免费** - 零成本运行和维护

## 🔧 自动部署流程

### 每日自动运行
1. **生成日报** - 日报系统生成HTML内容
2. **提交到Git** - 自动提交到GitHub仓库
3. **触发部署** - GitHub Actions自动构建
4. **更新网站** - 内容实时更新到GitHub Pages

### 手动更新
```bash
# 克隆仓库
git clone https://github.com/[你的用户名]/dora-ai-dream-archive.git

# 添加新日报文件
cp new-daily.html daily/ai/2026-03-21.html

# 提交更改
git add .
git commit -m "添加2026-03-21日报"
git push origin main
```
